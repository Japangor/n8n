# n8n resources & n8n.gjam.in

- **`site/`** — Astro static site for [n8n.gjam.in](https://n8n.gjam.in) (workflows, SEO guides, downloads)

## Deploy

See [site/README.md](site/README.md) for dev, build, and Vercel + domain setup.

cd site && npm install && npm run dev
